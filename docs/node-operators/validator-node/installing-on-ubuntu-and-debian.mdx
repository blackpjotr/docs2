# Installing on Ubuntu and Debian

Use this guide to install Mina packages on Ubuntu or Debian before generating keys or starting a validator node.

:::caution

If you already installed the standalone `mina-generate-keypair` package, remove it first:

```sh
sudo apt remove mina-generate-keypair
```

The keypair utility is included in `mina-mainnet` and `mina-devnet` packages.

:::

## Install Mina packages

<Tabs groupId="network">
<TabItem value="mainnet" label="Mainnet">

```sh
sudo rm /etc/apt/sources.list.d/mina*.list
echo "deb [trusted=yes] http://packages.o1test.net $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/mina.list
sudo apt-get update
sudo apt-get install --yes curl unzip mina-mainnet=3.3.0-8c0c2e6
```

</TabItem>
<TabItem value="devnet" label="Devnet">

```sh
sudo rm /etc/apt/sources.list.d/mina*.list
echo "deb [trusted=yes] http://packages.o1test.net $(lsb_release -cs) devnet" | sudo tee /etc/apt/sources.list.d/mina-devnet.list
sudo apt-get install --yes apt-transport-https
sudo apt-get update
sudo apt-get install --yes curl unzip mina-devnet=3.3.0-alpha1
```

</TabItem>
</Tabs>

## Verify your installation

```sh
mina version
```

The output should include a `Commit` line.
